# SUI testnet bot

## Getting started:

Clone this repository to yor machine:

```bash
https://github.com/jus1d/sui-testnet.git
```

Go to `sui-testnet` folder:

```bash
cd sui-testnet/
```

Install all dependencies via `npm`:

```bash
npm install
```

or `yarn`:

```bash
yarn install
```

Run one of commands:

```bash
npm run faucet
```

or other from `package.json`
